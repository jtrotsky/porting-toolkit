# <APP> — FreeBSD daemonless image

This repo builds a FreeBSD OCI image for [<app>](<upstream URL>) using `dbuild`.

## Quick reference

- `/port-package` — full porting workflow (research → build → CIT → PR)
- `/bump-upstream` — bump pinned upstream version and re-validate
- Cookbook: `.claude/reference/freebsd-porting-cookbook.md` — check here BEFORE guessing at errors
- Build notes: `BUILD-NOTES.md` — FreeBSD-specific changes and rationale

## Rules

- **Edit `Containerfile.j2` only** — the `Containerfile` is generated by `dbuild generate`
- **`dbuild build` exit 0 ≠ success** — grep the log for `error:`/`Failed`/`gyp ERR`
- **A green build ≠ a working image** — always run `dbuild test` (CIT)
- **Pin upstream release tags** — never build `main`
- **Branch off `upstream/main`** for PRs — keep the diff clean
- Run `scripts/lint-compose.sh` before committing to catch metadata mismatches
